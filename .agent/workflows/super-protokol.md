# Super Protokol v2

This workflow implements a rigid engineering cycle to ensure high-quality output.

## Phase 0: Skill Loading (Mandatory)
- Load required skills.
- Perform MCP health check.

## Phase 1: Environment Check
- Validate project workspace.
- Check configuration files.

## Phase 2: Strategy
- Define the exact plan.
- Resolve any ambiguity before starting.

## Phase 3: Planning
- Create sub-tasks.
- Request user confirmation for the plan.

## Phase 4: Engineering Cycle
- Implement using TDD.
- Validate each step.

## Phase 5: Finalization
- Git commit following conventions.
- Notify user of completion.
