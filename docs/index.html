Research Abstract

The Problem

Current digital systems rely on keyword indexing, which severs the semantic link between a Ruling (Hukm) and its Evidence (Dalil), leading to "Context Collapse".

The Solution

A Graph-based API that models knowledge as Nodes (Entities) and Edges (Relationships), preserving the Isnad and Derivation logic computationally.

The Stack

Built with Rust for type safety and SurrealDB for graph capabilities. Ingests data via a Neuro-Symbolic Python pipeline.
