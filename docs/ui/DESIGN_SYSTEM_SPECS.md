# UI Design System: "Noor" (Light)

> **Author**: UI Designer Agent
> **Date**: 2026-01-28
> **SFIA Alignment**: HCEV Level 5 (Advises on standards)

---

## 1. Design Tokens
### Typography (Inter & Amiri)
*   `font-primary`: 'Inter', sans-serif (UI).
*   `font-secondary`: 'Amiri', serif (Quranic Text).
*   `text-base`: 16px (1rem).

### Color Palette (Tawhidic Harmony)
*   `color-primary-500`: #0F766E (Teal - Wisdom).
*   `color-gold-500`: #D97706 (Gold - Divine).
*   `color-neutral-900`: #111827 (Ink).

## 2. Accessibility Standards (WCAG 2.1 AA)
*   **Contrast**: All text must meet 4.5:1 ratio.
*   **Touch Targets**: Minimum 44x44px for mobile interaction.
*   **Focus States**: Visible focus ring on all interactive elements.

## 3. Component Library
*   **Cards**: "Hadith Card", "Verse Card" (Atomic Units).
*   **Navigation**: "Graph Explorer" Sidebar (Collapsible).
*   **Visualization**: D3.js / Cytoscape canvas integration.
